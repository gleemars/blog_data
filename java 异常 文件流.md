## 异常
> 必检异常需要强制要求使用try-catch块进行处理, 例如 io 异常
> 必检异常在使用时需要先声明

### 异常声明
> 函数原型后面添加 异常声明

```java
public static func() throws exception1,exception2{
	// body
}

public static funcname()
throws exception1{
try{
	throws exception1;
}
catch (exception1 | exception2 ex){
	
}

finally{

}

}

```
